CSS

Important points:

  • Dashes between words in class and ID names. (kabob case)
  • Lowercase or uppercase colors are fine.
  • Order of properties is not important.
  • Separate selectors and declarations by new lines.
  • Use single quotation marks for attribute selectors and property values.
  • Omit quotes for URLs.
  • Please do not use LESS MQ convenience: put media queries at the bottom, and elements inside them.

Optional tags (thml, body, head) are required.
